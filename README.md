# PalmMuseum
BUCT-CS2301 - 掌上博物馆子系统

> BUCT-CS2301 团队 
> 海外藏中国文物知识管理与服务平台 · 移动端子系统

## 📱 项目简介

掌上博物馆是基于 **HarmonyOS** 开发的移动端 App，使用 ArkTS 语言与 ArkUI 框架。  
从知识图谱中选取单类文物（如瓷器/玉器/青铜器/中国画）作为展示主题，为用户提供沉浸式的文物浏览与互动体验。

主要功能包括：
- 文物浏览（卡片/瀑布流、详情查看、手势缩放）
- 语音导览（文物语音讲解、语音搜索、语音问答）
- 用户交互（点赞、收藏、评论、拍照上传）
- 以图搜图（相册/拍照上传，基于相似度检索）
- 个人信息管理（注册登录、个人主页、隐私设置）

## 🛠 技术栈

- **开发语言**：ArkTS
- **UI 框架**：ArkUI
- **运行平台**：HarmonyOS
- **数据来源**：知识图谱构建子系统（Neo4j / 三元组数据）
- **图像检索**：CLIP + FAISS（以图搜图）
- **语音能力**：HarmonyOS 语音 API
- **后端接口**：对接海外文物知识服务子系统
