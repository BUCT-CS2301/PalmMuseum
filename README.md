# PalmMuseum

BUCT-CS2301 - 掌上博物馆子系统

> 海外藏中国文物知识管理与服务平台 · 移动端子系统

## 📱 项目简介

掌上博物馆是基于 **HarmonyOS** 开发的移动端 App，使用 ArkTS 语言与 ArkUI 框架。  
本子系统不直接操作知识图谱，所有文物数据、问答服务、以图搜图等均通过调用后端（海外文物知识服务子系统、知识问答子系统）的 REST API 实现。

主要功能包括：
- 文物浏览（卡片/瀑布流、详情查看、多维度筛选、音视频播放）
- 语音导览（文物语音讲解、语音搜索、语音问答）
- 用户交互（点赞、收藏、评论、用户上传照片）
- 以图搜图（相册/拍照上传，后端完成特征比对与相似度检索）
- 个人信息管理（注册登录、个人主页、隐私设置）

## 🛠 技术栈

| 技术领域     | 技术选型                    |
| ------------ | --------------------------- |
| 开发平台     | HarmonyOS                   |
| 开发语言     | ArkTS                       |
| UI 框架      | ArkUI                       |
| IDE          | DevEco Studio 6.1.0 (Release) |
| 网络通信     | HTTP 协议                   |
| 本地存储     | Preferences / 关系型数据库  |
| 语音能力     | HarmonyOS 语音服务          |
| 以图搜图     | 相机 API + 图片选择器       |
| 音视频播放   | AVPlayer / Video 组件       |
| 图像加载     | Image 组件 + 图片缓存       |
| 版本控制     | Git + GitHub                |

> 说明：本子系统不直接操作知识图谱，所有业务数据通过后端 API 获取；以图搜图的特征提取与相似度检索由后端（CLIP+FAISS）完成，移动端仅负责图片采集与结果展示。

- DevEco Studio 6.1.0 (Release) 及以上
- HarmonyOS SDK API 12+
